SEIR
====

An OpenCOR-based Python script to model Covid-19 using the SEIR model.

Install
-------

From the ``Python Console`` window in `OpenCOR <https://opencor.ws/>`_::

 !pip install git+https://github.com/agarny/seir.git@master

Install
-------

From the ``Python Console`` window in `OpenCOR <https://opencor.ws/>`_::

 seir
